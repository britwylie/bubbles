# **Bubble Detector** 
## Purpose
First project using OpenCV in Python to track and count bubbles in a tube. 

**Hardware Required:**
* Raspberry Pi 2/3 B/B+
* PiCamera V2.1
* Neopixel LED strips
* 74AHCT125 level shifting chip
* box
* tube of bubbles

**Packages/Programs:**
* OpenCV
* fswebcam
* NeoPixel
* picamera
